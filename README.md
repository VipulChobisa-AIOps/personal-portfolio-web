## 🎨 Design Theme: Dark Luxury Glassmorphism

Designed to command authority and engage recruiters or clients instantly:
*   **Deep Contrast**: Solid `#050510` background overlayed with radial indigo and purple gradient accents.
*   **Glassmorphism Layouts**: Component cards styled with high-blur backdrops (`backdrop-filter`) and thin high-fidelity borders.
*   **Smooth Motion**: Transitions, micro-hover elevations, and slide-in animations.
*   **Responsive Framework**: Grid layout adapting dynamically to mobile, tablet, and widescreen monitors.

## 🛠️ Setup & Development

### 1. Install & Build
```bash
git clone https://github.com/VipulChobisa-AIOps/personal-portfolio-web.git
cd personal-portfolio-web
npm install
```

### 2. Run Local Development Server
```bash
npm run dev
# Starts server at http://localhost:3000
```

### 3. Build Production Bundle
```bash
npm run build
# Compiles into the /dist folder
```
